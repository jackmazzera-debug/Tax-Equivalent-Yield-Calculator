TEY math: Corporate = no adjustment. Treasury = yield / (1 - state rate). National Muni = yield / (1 - fed rate). State Muni = yield / (1 - fed - state). NIIT (3.8%) applies to treasury and taxable after-tax income when toggled on — muni income is NIIT-exempt. After-Tax $: Corporate = yield × (1 - fed - state) × portfolio. Treasury = yield × (1 - fed) × portfolio. National Muni = yield × (1 - state) × portfolio. State Muni = yield × portfolio. Yields from data.json refreshed nightly via GitHub Actions. Fallback values shown when live fetch unavailable. Not investment advice.
